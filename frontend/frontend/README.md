# LinkedIn Clone

A modern LinkedIn clone built with React and Material-UI.

## Features

- User Account & Authentication
- Profile Management
- Connections (Networking)
- News Feed & Posts
- Job Listings & Applications
- Search & Filtering
- Admin Panel

## Tech Stack

- React
- TypeScript
- Material-UI
- React Router
- Redux Toolkit
- Formik & Yup
- Axios

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd linkedin-clone
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application will be available at `http://localhost:3000`.
